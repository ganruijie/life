群聊：
  1. 1.打开群聊 → 点击群主分享的优惠券链接，进入优惠券详情页 → 点击领取
  2. 2.打开群聊 → 点击群主分享的二维码图片 → 保存图片到相册 → 打开【扫一扫】→ 添加图片 → 扫描图中二维码，进入优惠券详情页 → 点击领取
  3. 3.打开群聊 → 点击群主分享的二维码图片 → 长按图片，选择【识别图中二维码】，进入优惠券详情页 → 点击领取

单聊：
  1. 1.打开单聊 → 点击好友分享的优惠券链接，进入优惠券详情页 → 点击【加群领取】即可进群领取
  2. 2.打开单聊 → 点击好友分享的二维码图片，进入优惠券详情页 → 点击领取 → 保存图片到相册 → 打开【扫一扫】 → 添加图片 → 扫描图中二维码，进入优惠券详情页 → 点击【加群领取】即可进群领取
  3. 3.进入【okPay】->进去私聊->点击好友分享的二维码图片->长按图片->识别图中二维码->进去优惠券详情->点击加群领取->进群领取。
